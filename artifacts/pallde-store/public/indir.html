Mac Mini'de Açmak İçin:

  1. Dosyayı indir
  2. Terminal aç, bir klasör oluştur: mkdir pallde-store && cd pallde-store
  3. Arşivi aç: tar xf 25_Mart_2026_Full_Sistem_Kopyasi_Pallde_Store.tar
  4. Node.js kur (v20+): brew install node
  5. pnpm kur: npm install -g pnpm
  6. Bağımlılıkları yükle: pnpm install
  7. PostgreSQL kur ve .env dosyasına DATABASE_URL ekle
  8. Çalıştır: pnpm --filter @workspace/pallde-store run dev
